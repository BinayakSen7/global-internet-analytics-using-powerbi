# 🌍 Global Internet Usage Dashboard (Power BI)

An interactive Power BI dashboard analyzing global internet usage across countries. This project highlights internet penetration, total users, and the digital divide through clean and insightful visualizations.

---

## 📊 Project Overview

This dashboard provides a comprehensive view of worldwide internet usage, helping users explore patterns in connectivity and identify disparities between countries.

---

## 🎯 Key Features

* 🌍 **World Map Visualization** – Displays global distribution of internet users
* 📊 **Top Countries Analysis** – Highlights countries with the highest number of users
* 📈 **Internet Penetration Comparison** – Shows percentage of population using the internet
* 🥧 **User Share Distribution** – Visualizes each country’s contribution to global users
* 📋 **Detailed Table View** – Includes ranking, penetration rate, and user share

---

## 🧠 Key Insights

* Countries with large populations dominate total internet users
* High population does not always mean high internet penetration
* Significant digital divide exists between developed and developing regions

---

## 🧰 Tools & Technologies

* **Power BI Desktop** – Dashboard creation and visualization
* **DAX (Data Analysis Expressions)** – Measures and calculations
* **Power Query** – Data cleaning and transformation

---

## 📁 Dataset

* Source: Kaggle
* Dataset: *Global Internet Users by Country*
